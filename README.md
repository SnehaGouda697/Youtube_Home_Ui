Youtube Front home Clone Page
A frontend project that recreates the YouTube Home Page UI using HTML5 and CSS3. The goal of this project is to improve frontend development skills by practicing layouts, responsive design, and CSS Grid.

🚀 Features Implemented
Built the basic YouTube home page layout.
Added a search bar at the top of the page.
Created reusable video card components.
Displayed video thumbnails with channel profile images.
Added video title, channel name, view count, and upload time.
Used CSS Grid to create a multi-column video layout.
Styled the page using the Roboto font to closely match YouTube's design.
Implemented consistent spacing, typography, and circular channel profile images.
🛠️ Technologies Used
HTML5
CSS3
CSS Grid
Google Fonts (Roboto)
